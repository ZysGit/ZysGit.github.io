Android 环境配置说明

打包前请确认环境是否按照下面的步骤配置完成，如果机器是首次打Android包，由于涉及较多配置比较容易出错，可以让打包组同事协助处理。

工具版本及下载地址

名称 版本 下载方式
JDK JDK8 \192.168.0.250\常用软件\开发工具\android_tools\jdk
AndroidStudio 2020.3.1 https://developer.android.com/studio
NDK Unity：r19c
Cocos：r15c		 https://developer.android.com/ndk/downloads/older_releases
AndroidSdk 可以不用下载，在执行Gradle命令时会自动下载
Gradle 6.5+ https://developer.android.com/studio （可以不用下载，在执行Gradle命令时会自动下载）

环境变量配置（系统变量）

  1. JAVA_HOME=JDK安装根目录
  2. CLASSPATH=.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\toos.jar
  3. path 末尾追加 %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;
